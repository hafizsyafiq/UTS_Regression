# Load library
library(ggplot2)
library(corrplot)
## Warning: package 'corrplot' was built under R version 4.4.3
## corrplot 0.95 loaded
library(readr)

# Baca data
HeatingOil <- read.csv("HeatingOil.csv")
HeatingOil_Marketing <- read.csv("HeatingOil-Marketing.csv")

# Ambil variabel yang relevan dari HeatingOil, bukan 'data'
data_corr <- HeatingOil[, c("Heating_Oil", "Insulation", "Temperature", "Num_Occupants", "Avg_Age", "Home_Size")]

# Buat correlation matrix
cor_matrix <- cor(data_corr, use = "complete.obs")

# Tampilkan matrix korelasi dalam bentuk tabel
print(round(cor_matrix, 2))

## Home_Size          1.00
# Visualisasi: Heatmap
corrplot(cor_matrix, method = "color", type = "upper", tl.col = "black", tl.cex = 0.8)

Penjelasan:

Temperature (-0.75): Korelasi negatif kuat semakin hangat suhu di luar, semakin sedikit minyak pemanas digunakan.

Insulation (-0.60): Korelasi negatif cukup kuat rumah dengan insulasi yang baik butuh lebih sedikit minyak pemanas.

Home_Size (0.55): Korelasi positif rumah yang lebih besar cenderung membutuhkan lebih banyak minyak.

Num_Occupants (0.45) dan Avg_Age (0.30) juga memiliki korelasi positif, namun lebih lemah.

##Kesimpulan: Faktor-faktor yang paling berpengaruh terhadap konsumsi minyak adalah suhu udara dan kualitas insulasi. Rumah lebih besar juga cenderung lebih boros.

# Scatter plot

Penjelasan:

Terlihat bahwa tidak ada pola yang sangat kuat antara usia penghuni dengan konsumsi minyak.

Namun, kelompok usia menengah hingga tua mungkin sedikit lebih boros.

Penjelasan:

Titik hijau (konsumsi rendah) cenderung di suhu tinggi dan insulasi baik.

Strategi Pemasaran:

Fokus pada rumah yang berada di daerah bersuhu rendah berikan promosi spesial menjelang musim dingin.

Segmentasi rumah dengan insulasi rendah tawarkan paket hemat atau layanan konsultasi efisiensi energi.

Targetkan rumah besar rumah dengan ukuran besar cenderung konsumsi tinggi, bisa menjadi pelanggan utama.

Usia penghuni buat kampanye berbeda untuk kelompok usia menengah dan lansia (misalnya, layanan pengisian otomatis atau reminder SMS).

Gunakan pendekatan prediktif kembangkan sistem rekomendasi atau prediksi berbasis model regresi.
